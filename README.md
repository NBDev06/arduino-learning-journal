# Arduino Learning Journal

A place to organize my Arduino learning and get comfortable with GitHub.

There's a lot of very simple stuff here, but the main goal is to learn how to properly document my projects and create an archive of what I've done.

I write the notes myself, but I use AI for the .md formatting. It's because I prioritize efficiency—I'm currently doing a project everyday and it would quickly become burdensome to constantly write new README files from scratch.

## What You'll Find Here

- Chapter-by-chapter projects following a PDF guide
- Circuit diagrams and wiring notes
- Code with explanations in my own words
- Photos of my actual builds
- Notes on what I learned from each project
